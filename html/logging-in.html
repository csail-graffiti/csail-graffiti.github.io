Logging In

Graffiti apps are read-only unless you log in. To log in or out of Graffiti from a Vue app all you need to do is call $graffiti.toggleLogIn function. $graffiti is in the global Vue namespace so you can use it directly in templates or via this.$graffiti in components. You can log in from any domain.

To check whether a user is logged in, you can use the $graffitiMyActor variable. This variable is equal to a user's unique URI if they are logged in and null otherwise.
