Guide: Introduction

This guide explains how to build these web apps with the Graffiti client Javascript library and it's plugin for the Vue.js web framework.

If you haven't yet already, it's a good idea to read through the high level overview of the system.

If you're not familiar with Vue, make sure to check out the Vue docs before getting started. Note that there is nothing inherently tying Graffiti to Vue, it just happens to be the first framework we added support for. If you're interested in make a Graffiti plugin for a different web framework, please contact us or make a pull request.
