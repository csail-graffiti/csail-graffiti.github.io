Graffiti

is a project that aims to democratize the design of social applications by providing:

  1. A backend that implicitly resists lock-in
  2. A frontend framework that makes creating a social application almost as easy as creating a static website

To see Graffiti in action, check out the interactive tutorial on building Graffiti applications or the following example applications:

  • Namebook: kind of like Facebook.
  • Coeditor: kind of like Google Docs.
  • Activity Log: a utility for backing up, deleting and restoring Graffiti data.

Graffiti and all example applications are open source on Github.
