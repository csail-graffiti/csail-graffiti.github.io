Graffiti

is a project that aims to democratize the design of social applications by providing:

  1. A backend that implicitly resists vendor lock-in
  2. A frontend framework that makes creating a social application almost as easy as creating a static website

To see Graffiti in action, check out:

  • An interactive tutorial on building Graffiti applications
  • A Facebook clone built on top of Graffiti
  • A Google Docs clone built on top of Graffiti

Graffiti and the example applications are open source on Github.
