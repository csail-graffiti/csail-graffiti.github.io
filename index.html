Graffiti

is a project that aims to democratize the design of social applications by

  1. designing a backend that is implicitly immune to vendor lock-in and
  2. providing frontend tools that make creating a social network as easy as writing a static website.

  • examples of applications built with Graffiti → check out the gallery
